# Układarka puzzli

## Skład zespołu
- Mikołaj Walkowiak
- Paweł Przybyłowski
- Bartosz Ptak

## Przykładowe użycie programu:
* układamy puzzle do góry obrazkiem,
* robimy zdjęcie,
* program adnotuje puzzle na zdjęciu (np. oznacza je unikatowym numerem),
* generuje obraz i docelowe rozmieszczenie puzzli.
